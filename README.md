# Recruitment task

Project is created with WP 6.1 and Sage 9

## Installation

- Install WP 6.1
- Replace wp-content/themes && wp-content/uploads
- Add ACF PRO plugin
- Replace database using krovea.sql
- In folder theme/kroevea lunch commend:
  ```bash
      composer install
  ```
